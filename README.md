# 💼 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This portfolio showcases my technical skills, featured projects, certifications, and contact information with a clean developer-inspired interface.

## 🌟 Features

- Responsive design for desktop, tablet, and mobile
- Developer-themed UI with terminal-style hero section
- Smooth scrolling and scroll reveal animations
- Interactive typing animation
- Custom cursor effect
- Mobile-friendly navigation menu
- Skills, Projects, Certifications, and Contact sections
- Modern UI with gradient accents and clean typography

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts (Inter & JetBrains Mono)
